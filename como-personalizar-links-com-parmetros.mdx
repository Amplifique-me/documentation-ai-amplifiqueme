## Visão geral

O link de pesquisa da Amplifique.me aceita **parâmetros de URL** que permitem pré-identificar o respondente e enviar campos customizados automaticamente. Isso elimina o passo de preenchimento de dados e permite rastrear informações contextuais da resposta.

O padrão base do link é:

```
https://surveys.amplifique.me/survey/{ID_DA_PESQUISA}?ref={Nome_do_Link}
```

---

## Parâmetros de identificação do contato

| Campo | Parâmetro | Formato / Exemplo |
|---|---|---|
| Celular | `phone` | `55DD9XXXXXXXX` (ex: `5511999999999`) |
| E-mail | `email` | `email@example.com` |
| Nome | `name` | `Fulano da Silva` |

<Callout kind="info">
  Se apenas o nome for enviado (sem e-mail ou telefone), a pesquisa ainda solicitará ao respondente que informe ao menos um dos dois para identificação.
</Callout>

---

## Campos customizados

Campos customizados também podem ser passados via URL — tanto em pesquisas identificadas quanto anônimas.

**Padrão:**
```
customFields[NOME_INTERNO_DO_CAMPO]=VALOR
```

---

## Regras de sintaxe

- O **primeiro parâmetro** do link usa `?` como separador (por padrão é o `ref`)
- **Demais parâmetros** são separados por `&`
- Espaços e caracteres especiais nos valores devem ser codificados em URL (ex: espaço → `%20`)

---

## Exemplos práticos

**Apenas o celular:**
```
https://surveys.amplifique.me/survey/646ca2c79118f9ca6952578c?ref=Link1&phone=5511999999999
```

**Apenas um campo customizado:**
```
https://surveys.amplifique.me/survey/646ca2c79118f9ca6952578c?ref=Link1&customFields[tipo_cliente]=master
```

**Múltiplos campos + celular:**
```
https://surveys.amplifique.me/survey/646ca2c79118f9ca6952578c?ref=Link1&customFields[tipo_cliente]=master&phone=5511999999999&customFields[id_usuario]=12345
```

## Perguntas frequentes

<ExpandableGroup>

<Expandable title="Campos customizados passados via URL são salvos no cadastro do contato?">
  Sim. Os valores enviados via parâmetro são registrados tanto na resposta quanto no perfil do contato na plataforma, se o campo customizado já estiver criado nas configurações da conta.
</Expandable>

<Expandable title="Posso usar parâmetros de URL em pesquisas anônimas?">
  Sim. Campos customizados (`customFields`) funcionam tanto em pesquisas identificadas quanto anônimas. Parâmetros de identificação (`name`, `email`, `phone`) só fazem sentido em pesquisas identificadas.
</Expandable>

<Expandable title="Qual é o número máximo de campos customizados que posso passar na URL?">
  Não há um limite rígido documentado, mas URLs muito longas podem ser truncadas por alguns servidores ou clientes de e-mail. Para muitos campos, considere usar a API de disparo ao invés de parâmetros de URL.
</Expandable>

</ExpandableGroup>
