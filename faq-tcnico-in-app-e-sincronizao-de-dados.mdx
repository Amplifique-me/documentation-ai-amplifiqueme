## Visão geral

Este artigo responde as dúvidas técnicas mais comuns sobre o funcionamento do In-App na Amplifique.me, com foco em cenários de múltiplos idiomas, criação automática de usuários e sincronização de dados via script.

## Perguntas frequentes

<ExpandableGroup>

<Expandable title="Como criar pesquisas multilíngues no In-App?">
  No In-App, o espaço de tela é limitado — colocar todos os idiomas na mesma pergunta prejudica a experiência do usuário.

  **Recomendação:** crie uma pesquisa separada para cada idioma e use um campo customizado (ex: `idioma`) para controlar qual pesquisa será exibida para cada usuário.

  **Como funciona:**
  1. No seu sistema, envie o campo `idioma` dentro do script do In-App (ex: `PT`, `EN`, `ES`)
  2. Na Amplifique.me, configure cada pesquisa para ser exibida somente para o idioma correspondente usando as regras de elegibilidade

  Assim, o script é único, mas cada usuário vê a pesquisa no idioma correto conforme o valor do campo enviado.
</Expandable>

<Expandable title="Um novo usuário que acessa meu sistema é criado automaticamente na Amplifique.me?">
  Sim. Todo usuário que carregar o script do In-App será automaticamente verificado na base da Amplifique.me:

  - O sistema usa **e-mail ou telefone** para identificar se o usuário já existe
  - Se **não existir**: será criado automaticamente como novo contato
  - Se **já existir**: os dados serão atualizados conforme os campos enviados no script

  Isso garante que a base de contatos da Amplifique.me esteja sempre sincronizada com os usuários ativos do seu sistema.
</Expandable>

<Expandable title="Se um campo customizado for alterado no meu sistema, ele também muda na Amplifique.me?">
  Sim. Sempre que o script do In-App for carregado com novos dados, os campos customizados são **atualizados automaticamente** na Amplifique.me.

  **Importante:** a atualização acontece no momento em que o usuário acessa seu sistema e o script é executado — não há sincronização em background entre as sessões.
</Expandable>

<Expandable title="Qual é o tempo de sincronização dos dados?">
  A sincronização dos campos customizados ocorre em **tempo real**, no instante em que o script do In-App é executado. Não há atraso ou fila de processamento.
</Expandable>

<Expandable title="Preciso implementar o script novamente ao criar uma nova pesquisa In-App?">
  Não. O script é implementado **uma única vez** por unidade de negócio. As regras de qual pesquisa aparece para qual usuário são gerenciadas diretamente nas configurações de cada pesquisa dentro da plataforma — sem necessidade de alterações no código do seu sistema.
</Expandable>

</ExpandableGroup>
