## Visão geral

A Amplifique.me disponibiliza uma **API REST** completa para integração programática com qualquer sistema — ERPs, CRMs, plataformas customizadas ou qualquer outra aplicação que precise disparar pesquisas ou gerenciar dados sem passar pela interface da plataforma.

## Acesso pela plataforma

Você também pode acessar a documentação diretamente pelo menu da plataforma:

No menu lateral, clique nos **três pontos (`⋯`)** → **Integrações** → botão **Ver Integração** ao lado de qualquer integração listada.

---

## Como autenticar

Todas as requisições à API utilizam um **token de API** gerado dentro da plataforma em **Integrações → API**. O token é passado como parâmetro na URL no formato:

```
?access_token=Bearer+SEU_TOKEN_AQUI
```

<Callout kind="alert">
  Trate o token como uma senha. Não o exponha em repositórios públicos, logs ou URLs acessíveis publicamente.
</Callout>
