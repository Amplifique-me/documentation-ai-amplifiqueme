title: Como exportar respostas de pesquisas?
description: >-
  Exporte respostas finalizadas, não finalizadas ou solicitações de uma pesquisa
  em formato Excel. Acompanhe o progresso na fila de exportação e receba a
  planilha por e-mail.
---
## Visão geral

A exportação gera uma planilha Excel (.xlsx) com todas as respostas da pesquisa selecionada. Você pode filtrar o tipo de dado antes de exportar e acompanhar o progresso na fila de exportações.

---

## Como exportar

<Steps>
  <Step title="Acesse Pesquisas e Respostas" title-type="p">
    No menu lateral, clique em **Pesquisas e Respostas** e acesse a aba **Pesquisas**.
  </Step>

  <Step title="Localize a pesquisa e clique em Exportar" title-type="p">
    Passe o mouse sobre a pesquisa desejada — alguns ícones de ação serão exibidos. Clique no ícone **Exportar**.\


    <Image src="https://blob-cdn.documentation.ai/org-228168c3-baff-41c0-9e3d-a927017afd0f/doc-dc08e65a-c672-46a6-8e86-3eb57bb5df19/1774987397295-djpjtr1i4j9-pasted-image-1774987395071.png?q=85&fm=auto&auto=compress%2Cformat" width="1556" height="827" alt="" />

    <Callout kind="tip" collapsed="false">
      Se houver várias pastas de pesquisas, localize a pasta correta antes de buscar a pesquisa.
    </Callout>
  </Step>

  <Step title="Escolha o tipo de exportação" title-type="p">
    Selecione uma das opções disponíveis:

    | Opção                                  | O que exporta                                          |
    | -------------------------------------- | ------------------------------------------------------ |
    | **Exportar tudo**                      | Todas as respostas, independentemente do status        |
    | **Exportar respostas finalizadas**     | Apenas respostas em que o respondente clicou em enviar |
    | **Exportar respostas não finalizadas** | Respostas abertas mas não submetidas                   |
    | **Exportar solicitações**              | Histórico de disparos (envios realizados)              |
  </Step>

  <Step title="Acompanhe o progresso" title-type="p">
    Após iniciar, uma mensagem aparece no final da página. Clique nela para ir direto à **fila de exportação**.

    Você também pode acessar a fila pelo menu: três pontos (`⋯`) → **Exportações**.\


    <Image src="https://blob-cdn.documentation.ai/org-228168c3-baff-41c0-9e3d-a927017afd0f/doc-dc08e65a-c672-46a6-8e86-3eb57bb5df19/1774987444015-j1sjxxx5da-pasted-image-1774987441799.png?q=85&fm=auto&auto=compress%2Cformat" width="1557" height="831" alt="" style="width: 1557px; height: auto; cursor: pointer;" />

    <Callout kind="success" collapsed="false">
      A planilha exportada também é **enviada automaticamente para o seu e-mail** assim que o processamento for concluído.
    </Callout>
  </Step>
</Steps>

## Entendendo as colunas da planilha exportada

A planilha gerada contém colunas fixas de sistema e colunas dinâmicas das suas perguntas. Veja o significado das colunas fixas:

| Coluna                | Descrição                                                               |
| --------------------- | ----------------------------------------------------------------------- |
| **ID**                | Código único de cada resposta                                           |
| **LINK**              | Link direto para a resposta na plataforma                               |
| **SURVEY**            | Nome da pesquisa                                                        |
| **ANONYMOUS**         | Indica se a pesquisa é totalmente anônima                               |
| **SENT\_AT**          | Data em que a pesquisa foi enviada                                      |
| **OPENED\_AT**        | Data da última abertura pelo respondente                                |
| **FINALIZED\_AT**     | Data em que a resposta foi finalizada (enviada)                         |
| **FINALIZED**         | `true` se o respondente clicou em enviar; `false` se não finalizou      |
| **OPENED**            | `true` se a pesquisa foi aberta                                         |
| **CHANNEL**           | Canal de envio (ver tabela abaixo)                                      |
| **ORIGIN**            | Origem: link/QR Code, InApp ou API                                      |
| **TIME\_TO\_ANSWER**  | Tempo total que o respondente levou para responder                      |
| **CUSTOMER\_NAME**    | Nome do respondente                                                     |
| **CUSTOMER\_EMAIL**   | E-mail do respondente                                                   |
| **CUSTOMER\_PHONE**   | Telefone do respondente                                                 |
| **CUSTOMER\_COMPANY** | Empresa do respondente                                                  |
| **SOURCE\_STATUS**    | Status do envio: `Sent`, `Error`, `Pending` ou `Desconhecido` (Link/QR) |
| **SENDING\_ERROR**    | Detalhe do erro de envio, quando houver                                 |

**Valores da coluna CHANNEL:**

| Código | Canal                  |
| ------ | ---------------------- |
| 0      | SMS                    |
| 1      | E-mail                 |
| 2      | WhatsApp Web           |
| 3      | Link / QR Code / Totem |
| 4      | InApp                  |
| 6      | WhatsApp API (Oficial) |
| 8      | WhatsApp Flows         |

## Perguntas frequentes

<ExpandableGroup>
  <Expandable title="Por que uma resposta aparece como não finalizada?" default-open="false">
    O respondente abriu a pesquisa mas não clicou no botão de enviar. Use a opção "Exportar respostas não finalizadas" para analisar esse grupo separadamente.
  </Expandable>

  <Expandable title="A planilha inclui as respostas das perguntas abertas e as notas?" default-open="false">
    Sim. Além das colunas fixas de sistema, a planilha inclui uma coluna para cada pergunta da pesquisa.
  </Expandable>

  <Expandable title="Posso exportar respostas de um período específico?" default-open="false">
    A exportação não tem filtro de data diretamente — ela exporta todas as respostas da pesquisa. Para analisar por período, utilize os filtros de data na aba Respostas antes de exportar, ou filtre a planilha no Excel após o download.
  </Expandable>

  <Expandable title="Quanto tempo demora para a exportação ficar disponível?" default-open="false">
    Depende do volume de respostas. Pesquisas com muitas respostas podem levar alguns minutos. Acompanhe o progresso na fila de exportações e aguarde o e-mail de confirmação.
  </Expandable>
</ExpandableGroup>
